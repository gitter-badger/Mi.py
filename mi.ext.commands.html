run(uri, token, debug=False)[ソース]¶

Launch the bot. :param uri: websocket url of the Misskey instance to connect to :type uri: str :param token: Misskey account token :type token: str :param debug: Debug Mode :type debug: bool

サンプル

When inheriting from a class:

class MyBot(Bot):
    async def on_message(self, ws, message):
        pass
bot = MyBot()
bot.run(uri, token)

When using a listener:

bot = Bot()

@bot.event()
async def on_message(ws, message):
    pass

bot.run(uri, token)
戻り値

None

戻り値の型

None
